# Heart Disease Prediction Project

## Overview
This project aims to predict the presence of heart disease using patient data and machine learning techniques. The goal is to identify key risk factors and build a model that can support early detection and decision-making in healthcare settings.

## Dataset
The dataset used in this project is the UCI Heart Disease dataset. It includes both demographic and clinical features such as age, sex, chest pain type, cholesterol levels, and maximum heart rate.

## Project Workflow
The project follows a standard data analysis and modeling process:

- Data cleaning and preprocessing  
- Exploratory Data Analysis (EDA)  
- Feature preparation  
- Model training and evaluation  
- Business insights and recommendations  

## Models Used
Two models were implemented and compared:

- Logistic Regression  
- Decision Tree  

Logistic Regression achieved better performance with an accuracy of approximately 61.7%, while Decision Tree achieved around 53.3%.

## Key Findings
- Age and chest pain type are strong indicators of heart disease  
- Logistic Regression provided more stable and interpretable results  
- Simpler models can sometimes outperform more complex ones
